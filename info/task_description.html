Do you know how to play in Prime Palindrome Golf? You are given a number and you should find the closest palindromic prime number that greater than the given.

A palindromic number or numeral palindrome is a number that remains the same when its digits are reversed. Like 79197, for example, it is "symmetrical".
A prime number is a natural number greater than 1 that has no positive divisors other than 1 and itself. In this task you will be given an positive integer. You should find the closest integer that:
- is greater than the given number;
- is prime;
- is palindromic.
For example, the number is 13 and the closest greater palindromic prime is 131. One more example, the number is 2 and the answer is 3, because any one-digit number is a palindrome.

Input: A number as an integer.

Output: The closest greater palindromic prime as an integer.
